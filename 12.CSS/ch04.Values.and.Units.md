# 4. Values and Units

## 4.1 Keywords, Strings, and Other Text Values

### 4.1.1 Keywords

例如，关键字 none 例子：移除链接的下划线：

    a:link, a:visited {
      text-decoration: none;
    }

与去掉下划线相反，要增加下划线，则使用关键字：

    underline

同一个关键字，用在不同的属性上，表达的意思也许完全不同，例如下面两个 normal 表达的是不同的意思：

    div {
      letter-spacing: normal;
      font-style: normal;
    }
